About Us

Welcome to Aether Wings Airways, a comprehensive full-stack application designed to simplify the process of booking flights. This project leverages modern technologies and best practices in software development to provide a robust, secure, and user-friendly experience.

Key Features:

  • Backend: The backend of this project is built using Java Spring Boot and follows a microservices architecture. Each service is designed to handle specific business functionalities, ensuring scalability and maintainability.
  • Database: We utilize MongoDB and MySQL for data storage, leveraging Spring Data JPA for efficient data management. MongoDB is used for handling large sets of unstructured data, while MySQL is used for structured data requirements.
  • Security: To secure our backend services, we implement Spring Security. All API calls are routed through an API Gateway, ensuring that our system is protected against unauthorized access.
